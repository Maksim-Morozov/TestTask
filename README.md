# 📋 Kanban Dashboard

Современная канбан-доска для управления задачами с Drag & Drop, приоритетами и цветовыми метками.

## 🚀 Деплой проекта **(https://agent-69eab52c0d8a88c--precious-rabanadas-dbdb46.netlify.app/)**

## ✨ Возможности

- 📝 Создание, редактирование и удаление карточек задач
- 📊 Drag & Drop перемещение карточек между колонками
- 🎨 Изменение цвета колонок
- 🏷️ Приоритеты задач (Low / Medium / High)
- 💾 Автоматическое сохранение в LocalStorage
- 📱 Адаптивный дизайн
- ✨ Плавные анимации

## 🛠️ Технологии

- React 18
- TypeScript
- Redux Toolkit
- Styled Components
- Vite
